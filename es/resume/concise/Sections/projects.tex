%============ Proyectos ============%
\section*{Proyectos Seleccionados}

\begin{itemize}[leftmargin=12pt]
    \item \textbf{ari64} — Biblioteca C de enteros grandes enfocada principalmente en implementaciones criptográficas y aritmética de precisión arbitraria.
    \item \textbf{RetailBase.Identity} — Módulo de autenticación y gestión de usuarios con registro seguro, recuperación y flujos empresariales basados en JWT.
    \item \textbf{RetailBase} — Base modular para aplicaciones empresariales con bloques backend reutilizables y arquitectura en capas.
    \item \textbf{diagrams} — Proyecto ligero en HTML para publicar diagramas técnicos y de arquitectura.
    \item \textbf{HookHubNet} — Túneles TCP seguros sobre WebSockets con hub centralizado y reenvío dinámico de puertos.
    \item \textbf{Plataformas Cryoinfra} — Sistemas web y móviles (.NET Core, .NET MAUI) para logística, monitoreo de producción y operaciones internas.
    \item \textbf{Arquitectura Cryoinfra} — Definición de arquitectura en capas, contratos de servicio y estándares de desarrollo.
    \item \textbf{Sistema de Mesa de Ayuda} — Plataforma interna de tickets con soporte asistido por IA (Watson AI).
    \item \textbf{Hard.Core} — Framework empresarial modular que acelera el desarrollo de aplicaciones.
    \item \textbf{CI/CD Privado} — Pipelines autoalojados usando Jenkins, Docker y Ansible.
    \item \textbf{HookHub} — Hub de mensajería en tiempo real usando ASP.NET Core y SignalR.
    \item \textbf{Midori64 / Midori128} — Implementaciones ligeras de cifradores de bloque criptográficos.
\end{itemize}
